# Medical Economics Performance Dashboard

## Power BI | Healthcare Claims Analytics | 2020–2021

### Project Overview
This Power BI dashboard analyzes healthcare claims data across two fiscal years (2020 and 2021), providing stakeholders with a comprehensive view of medical spending, claim volume, provider performance, regional distribution, and diagnosis cost drivers.
The report is designed for medical economics and operations teams to monitor ER, PCP, Specialist, and Urgent Care visit trends — with dynamic narrative context that updates automatically based on slicer selections.

###  How to Use
1. Clone or download this repository
2. Open Healthcare_Claims_Dashboard.pbix in Power BI Desktop
3. If prompted, update the data source path to point to your local claims_data file
4. Use the Visit Type buttons (ER / PCP / Specialist / Urgent Care) to filter the view
5. Use the Year slicer to switch between 2020 and 2021

Note: Power BI Desktop is free to download at powerbi.microsoft.com
